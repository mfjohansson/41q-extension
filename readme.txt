Dot 41q on pages

== Description ==

This plugin adds .41q URL extension to the 41q site

== Powered By ==
Raketforskning (http://www.41q.com)

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Appends .41q to the URL of pages when using permalinks.
